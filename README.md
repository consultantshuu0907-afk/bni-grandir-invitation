# BNI GRANDIR 特別招待状ページ

BNI GRANDIRチャプターのBusiness Open Dayへ案内する、深紅×ゴールド基調の静的ランディングページです。封筒をタップして開封する演出、開催情報、カウントダウン、Googleフォームへの申込導線を備えています。

---

## ✅ 現在実装済みの機能

- **封筒タップアニメーション**
  - 封筒をクリック／タップするとフラップが開き、便箋が表示されます。
  - キーボード操作にも対応しています。
- **リアル封蝋ビジュアル**
  - `images/realistic-wax-seal-g.svg` にローカルSVG画像として配置。
  - 外部生成画像URLに依存しないため、安定表示できます。
- **高級感のある赤×金デザイン**
  - ボルドー封筒、ゴールド装飾、背景パーティクルを使用。
  - 背景・封筒・金文字・ボタンに多層グラデーションとハイライト／沈み影を追加し、色の立体感を強化。
  - 封筒本体を暗めに沈め、封蝋・金文字・CTAボタンが前面に浮いて見えるようコントラストを再調整。
- **カウントダウンタイマー**
  - 2026年7月3日（金）AM 9:15 に向けて残り時間を表示。
- **本文段落の整形**
  - ABOUT BODと最下部CTAの文章を読みやすい段落・改行・余白に調整。
  - スマホでは不自然な固定改行を抑制。
- **スクロール演出**
  - コンテンツが表示領域に入るとフェードアップします。
- **Googleフォームへの申込導線**
  - 申込ボタンから外部Googleフォームへ遷移します。
- **レスポンシブ対応**
  - モバイル表示に合わせて余白・文字組み・ボタンサイズを調整。
- **モーション配慮**
  - `prefers-reduced-motion` に対応しています。
